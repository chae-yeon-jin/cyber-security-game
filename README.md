# IT Security Gamification: Hacker & Defender Simulation

> "Know your enemy and know yourself, and you will never be defeated."

## 🎮 Project Overview
[cite_start]This project is a **web-based interactive simulation** consisting of 5 missions designed to help players experience the core principles of cyber-attack and defense[cite: 70]. [cite_start]Players take on the role of a new intern in a fictional hacker group, tasked with penetrating a company with weak security measures[cite: 30]. [cite_start]This "attacker-centric" approach helps learners understand how different techniques are deployed in a real-world cybersecurity attack[cite: 31, 35].

## 🎯 Educational Objectives
* [cite_start]**Master Fundamental Concepts:** Gain an intuitive understanding of core cybersecurity principles and terminology[cite: 34].
* [cite_start]**Attacker-Centric Strategies:** Learn the logical flow of a cyberattack, from initial penetration to data exfiltration[cite: 35].
* [cite_start]**Defender-Centric Response:** Develop practical Incident Response skills, such as prioritizing assets and executing mitigation steps like network isolation[cite: 37].
* [cite_start]**Practical Security Governance:** Foster a managerial mindset for evaluating data value and applying appropriate security policies[cite: 38].

## 🕹️ Level Analysis
[cite_start]The game follows a structured journey from gaining a foothold to establishing defense governance[cite: 72, 73].

| Level | Name | Key Content & Learning Outcome |
| :--- | :--- | :--- |
| **Level 1** | **Hash Breaker** | [cite_start]Perform a **Dictionary Attack** to recover original passwords from leaked hashes[cite: 75, 76]. |
| **Level 2** | **Phishing Designer** | [cite_start]Design a convincing **Phishing Email** to understand the anatomy of social engineering[cite: 86, 87]. |
| **Level 3** | **Virtual Desktop** | [cite_start]Categorize target assets and execute **Incident Response** procedures after a Trojan infection[cite: 95, 96, 102]. |
| **Level 4** | **Escalation Sequence** | [cite_start]Arrange the logical sequence of an attack, including **Privilege Escalation** and **Keyloggers**[cite: 105, 110]. |
| **Level 5** | **Data Discovery** | [cite_start]Classify data sensitivity and apply **Security Controls** (Encryption, MFA, etc.) from a managerial perspective[cite: 112, 117]. |

## 💡 Key IT Security Concepts Covered
* [cite_start]**Hashing & Dictionary Attacks:** Transforming passwords into fixed-length strings and cracking them via pre-made lists[cite: 41, 44].
* [cite_start]**Social Engineering & Phishing:** Manipulating human behavior to steal sensitive information via deceptive emails[cite: 49, 52].
* [cite_start]**Malware (Trojan & Backdoor):** Disguised malicious programs and secret access points used to bypass security[cite: 54, 55].
* [cite_start]**Privilege Escalation:** Systematically upgrading from limited access to administrative rights[cite: 57, 58].
* [cite_start]**Data Exfiltration & Governance:** Unauthorized data transfer and the framework for protecting assets based on risk[cite: 63, 118].
