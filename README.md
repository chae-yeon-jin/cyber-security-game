# IT Security Gamification: Hacker & Defender Simulation

> "Know your enemy and know yourself, and you will never be defeated."

## 🎮 Project Overview
This project is a **web-based interactive simulation** consisting of 5 missions designed to help players experience the core principles of cyber-attack and defense. Players take on the role of a new intern in a fictional hacker group, tasked with penetrating a company with weak security measures. This "attacker-centric" approach helps learners understand how different techniques are deployed in a real-world cybersecurity attack.

## 🎯 Educational Objectives
* **Master Fundamental Concepts:** Gain an intuitive understanding of core cybersecurity principles and terminology.
* **Attacker-Centric Strategies:** Learn the logical flow of a cyberattack, from initial penetration to data exfiltration.
* **Defender-Centric Response:** Develop practical Incident Response skills, such as prioritizing assets and executing mitigation steps like network isolation.
* **Practical Security Governance:** Foster a managerial mindset for evaluating data value and applying appropriate security policies.

## 🕹️ Level Analysis
The game follows a structured journey from gaining a foothold to establishing defense governance.

| Level | Name | Key Content & Learning Outcome |
| :--- | :--- | :--- |
| **Level 1** | **Hash Breaker** | Perform a **Dictionary Attack** to recover original passwords from leaked hashes. |
| **Level 2** | **Phishing Designer** | Design a convincing **Phishing Email** to understand the anatomy of social engineering. |
| **Level 3** | **Virtual Desktop** | Categorize target assets and execute **Incident Response** procedures after a Trojan infection. |
| **Level 4** | **Escalation Sequence** | Arrange the logical sequence of an attack, including **Privilege Escalation** and **Keyloggers**. |
| **Level 5** | **Data Discovery** | Classify data sensitivity and apply **Security Controls** (Encryption, MFA, etc.) from a managerial perspective. |

## 💡 Key IT Security Concepts Covered
* **Hashing & Dictionary Attacks:** Transforming passwords into fixed-length strings and cracking them via pre-made lists.
* **Social Engineering & Phishing:** Manipulating human behavior to steal sensitive information via deceptive emails.
* **Malware (Trojan & Backdoor):** Disguised malicious programs and secret access points used to bypass security.
* **Privilege Escalation:** Systematically upgrading from limited access to administrative rights.
* **Data Exfiltration & Governance:** Unauthorized data transfer and the framework for protecting assets based on risk.
